―――――
「本日の運勢」の編集したい場合
―――――
* 「data.json」を編集します。
* 使える運勢ランクは「大吉／吉／中吉／小吉／末吉／凶」です。デフォルトでは「大吉／吉／中吉」になっています。
* 同じランクを複数登録すると、その分だけ出やすくなります（例：「大吉」を3行にすれば大吉が出る確率がアップ）。

―――――
「本日の開運スポット（自治体名）」を編集したい場合
―――――
* 「local.csv」を編集し、自治体名を追加または削除してください。
* 自治体名を追加した場合、後ろに必ず「,（半角カンマ）」を付けてください。
* 1行目は削除しないでください。

―――――
「開運アクション」を編集したい場合
―――――
* 「action.csv」を編集し、開運アクションを追加または削除してください。
* 新たに追加する場合、「{action},{question}」のように「,（半角カンマ）」で区切ってください（例：地域への愛を聞いてみよう,「この地域を好きな理由はなんですか？」）
* 1行目は削除しないでください。
